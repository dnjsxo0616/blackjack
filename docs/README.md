# 풀이과정 및 코드 설명

# domain
- [ ] 플레이어 카드의 1~11 크기 중 무작위의 카드 한 장을 선택하는 기능
  - [ ] 선택한 카드를 플레이어가 뽑은 카드를 누적해서 저장하는 기능

- [ ] 딜러 카드의 1~11 크기 중 무작위의 카드 한 장을 선택하는 기능
  - [ ] 선택한 카드를 딜러가 뽑은 카드를 누적해서 저장하는 기능

- [ ] 선택된 플레이어의 카드와 딜러의 카드를 비교하는 기능
  - [ ] 숫자가 큰 사람을 승자
  - [ ] 같은 값이라면 무승부
  - [ ] 플레이어의 승패무를 저장하는 기능

- [ ] 게임을 계속 진행할지 중단할지 결정하는 기능
  - [ ] Y와 N만 입력받는다.
  - [ ] Y은 계속 진행
  - [ ] N은 종료 메시지와 함께 게임 중단

- [ ] 라운드를 카운트하는 기능
